原文：About the Language Reference

使用内置的命名和复合类型。

本书的这一部分描述了 Swift 编程语言的形式语法。这里描述的语法旨在帮助你更详细地理解语言，而不是让你直接实现解析器或编译器。

Swift 语言相对较小，因为在 Swift 代码中几乎无处不在的许多常见类型、函数和运算符实际上都定义在 Swift 标准库中。尽管这些类型、函数和运算符不是 Swift 语言本身的一部分，但它们在本书这一部分的讨论和代码示例中被广泛使用。

如何阅读语法​

用于描述 Swift 编程语言的形式语法的符号遵循一些约定：

  • 箭头 (→) 用于标记语法产生式，可以理解为"可以由...组成"。

  • 句法类别由斜体文本表示，并出现在语法产生式规则的两侧。

  • 文字词和标点符号由文本表示 boldface constant width，并且仅出现在语法产生式规则的右侧。

  • 替代语法产生式由竖线 (|) 分隔。当备选作品太长而难以阅读时，它们会在新行上分解为多个语法作品规则。

  • 在少数情况下，常规字体文本用于描述语法产生式规则的右侧。

  • 可选的句法类别和文字由尾随问号标记，？.

例如，getter-setter 块的语法定义如下：

getter-setter 块的语法
getter-setter-block → { getter-clause setter-clause? } | { setter-clause getter-clause }

此定义表明 getter-setter 块可以包含一个 getter 子句，后跟一个可选的 setter 子句，用大括号括起来，或者一个 setter 子句，后跟一个 getter 子句，用大括号括起来。上面的语法产生式等同于以下两个产生式，其中的替代项已明确说明：

getter-setter 块的语法
getter-setter-block → { getter-clause setter-clause? }

getter-setter-block → { setter-clause getter-clause }

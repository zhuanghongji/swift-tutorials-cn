## 概述

该类是 AVAsset 的具体子类。当你创建如下所示的资产时，系统会创建并返回 AVURLAsset 的实例。

```swift
let url: URL = // A local or remote asset URL.
let asset = AVAsset(url: url)
```

在许多情况下，这是创建资产实例的适当方法。但是，当你需要对其初始化进行更细粒度的控制时，你也可以直接实例化 AVURLAsset 。 AVURLAsset 的初始化程序接受一个选项字典，你可以使用该字典根据你的特定目的自定义资产的初始化。例如，如果你正在为 HLS 流创建资产，你可能希望阻止它在通过蜂窝网络连接时检索其媒体。你可以通过提供初始化选项和值来完成此操作，如下所示。

```swift
let url: URL = // A remote asset URL.
let options = [AVURLAssetAllowsCellularAccessKey: false]
let asset = AVURLAsset(url: url, options: options)
```

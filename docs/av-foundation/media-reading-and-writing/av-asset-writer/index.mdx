## 概述

你可以使用资产写入器将媒体写入文件格式，例如 QuickTime 影片文件格式和 MPEG-4 文件格式。资产写入器自动支持来自并发轨道的交错媒体数据，以实现高效播放和存储。它可以重新编码写入输出文件的媒体样本，还可以将元数据集合写入输出文件。

:::caution

资产写入器是一种写入一个输出文件的一次性对象。如果你的应用程序需要写入多个输出文件，请创建多个资产写入器实例。

:::

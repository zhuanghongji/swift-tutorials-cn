## 概述

创建资产写入器输入以将单个媒体轨道和可选的轨道级元数据写入输出文件。要以理想的媒体数据交错方式写入多个并发轨道，请观察每个输入的 isReadyForMoreMediaData 属性的值。

你可以使用资产编写器输入在 QuickTime 影片文件中创建非独立的轨道，而是引用另一个文件中存在的示例数据。

## 概述

资产可能包含多个媒体轨道，当你播放或处理它们时，这些轨道是互斥的。例如，一项资产可能包含不同口语的多个音轨，但一次只能播放其中一个。你可以使用输入组将轨道集合标记为在资产编写器输出的文件中彼此互斥。

:::note

通过调用 addTrackAssociation(withTrackOf:type:) 关联多个轨道后，你可以在写入文件之前检查资产编写器输出的媒体选择选项。

:::

## 概述

你可以将具体的输出实例（例如 AVAssetReaderTrackOutput 或 AVAssetReaderVideoCompositionOutput ）添加到资产读取器以执行特定任务。

:::caution

如果你不需要就地修改示例数据，请将 alwaysCopiesSampleData 属性的值设置为 false 以防止输出进行不必要的复制。

:::

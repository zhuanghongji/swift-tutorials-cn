:::

## 配置示例代码项目 in page link

在 Xcode 中运行示例代码项目之前：

- 编辑名为 fmp4Writer 的共享方案。
- 打开"运行"操作。
- 将 `<磁盘上电影文件的路径>` 参数替换为本地硬盘驱动器上电影文件的路径。
- 将 `<输出目录的路径>` 参数替换为所需的输出目录；例如 ~/Desktop/fmp4writer/ 。

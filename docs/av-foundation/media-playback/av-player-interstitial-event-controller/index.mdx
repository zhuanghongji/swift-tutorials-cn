## 概述

此类是 AVPlayerInterstitialEventMonitor 的子类，用于管理在主要内容播放期间呈现的插页式事件的计划。

:::caution

创建事件控制器并设置时间表会导致播放忽略源媒体中存在的间隙事件。

:::

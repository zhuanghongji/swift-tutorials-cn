## 概述

播放器项存储对 AVAsset 对象的引用，该对象表示要播放的媒体。如果你需要在将资源放入队列进行播放之前对其进行检查，请调用其 load(\_:) 方法来检索一个或多个属性的值。或者，你可以通过将所需属性传递给其 init(asset:automaticallyLoadedAssetKeys:) 初始值设定项来告诉播放器项目自动加载所需属性。当播放器项目准备好播放时，你请求的资产属性就可以使用了。

## 初始化器

### init(data:desiredCount:minimumStride:)

```swift
// 自动根据数据确定 bin
//
// - data: 给定的数据值
// - desiredCount: 给定数据所需的 bin 数量。如果 nil ，则使用 Scott’s normal reference rule
//                 自动从数据推断 bin 数量，上限为 200。
// - minimumStride: 允许的最小 bin 大小
// - Returns: 推断的 bins
public init(data: [Value],
            desiredCount: Int? = nil,
            minimumStride: Value = 0) where Value : BinaryFloatingPoint
```

### init(range:count:)

```swift
// 为范围创建给定数量的 bin
// 期望范围长度是 count 的倍数以允许统一的整数 bin
//
// - range: bin 的范围
//          第一个 bin 从范围的下限开始，最后一个 bin 在范围的上限结束
// - count: bin 的确切数量
public init(range: ClosedRange<Value>, count: Int) where Value : BinaryInteger
```

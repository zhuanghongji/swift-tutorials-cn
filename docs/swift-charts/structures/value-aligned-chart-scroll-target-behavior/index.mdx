## 初始化器

### init(matching:majorAlignment:limitBehavior:)

```swift
// 创建一个滚动目标行为，该行为与沿可滚动轴以规则间隔间隔的值对齐
//
// - components: 用户完成滚动后对齐时要搜索的组件
// - majorAlignment: 与主要价值观保持一致的行为。当用户在图表上滑动时，
//                   图表将根据滑动方向捕捉到下一个或上一个主要单位。
//                   启用后，默认主要单位是页。
// - limitBehavior: 滚动限制行为
public init(matching components: DateComponents,
            majorAlignment: MajorValueAlignment<Date>? = nil,
            limitBehavior: ValueAlignedLimitBehavior = .automatic)
```

### init(unit:majorAlignment:limitBehavior:)

```swift
// 创建一个滚动目标行为，该行为与沿可滚动轴以规则间隔间隔的值对齐
//
// - unit: 对齐单元。当用户完成滚动手势时，图表将对齐到给定单位或域的末尾
// - majorAlignment: 与主要价值观保持一致的行为。当用户在图表上滑动时，
//                   图表将根据滑动方向捕捉到下一个或上一个主要单位。
//                   启用后，默认主要单位是页。
// - limitBehavior: 滚动限制行为
public init<T>(unit: T,
               majorAlignment: MajorValueAlignment<T>? = nil,
               limitBehavior: ValueAlignedLimitBehavior = .automatic
) where T : Plottable, T : Numeric
```

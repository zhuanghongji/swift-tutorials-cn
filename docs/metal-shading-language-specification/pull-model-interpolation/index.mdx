# 拉模型插值

<OrigninalPDF title="6.11 Pull-Model interpolation" />

所有操作系统：自 Metal 2.3 起支持拉模型插值。

插值类型 `interpolant<T,P>` （第 2.18 节）和相关方法在 `<metal_interpolate>` 中定义。在片段函数中，你可以通过调用其方法来显式插值 `interpolant<T,P>` 类型的值，如下所示。插值可以以不同的模式被多次采样和插值，并且可以被传递到其它函数以在那里被采样和插值。透视正确性在参数的所有插值中由其类型中的 `P` 值固定。

Metal 支持：

- `f` 或 `F` 后缀指定单精度浮点文字值（例如 `0.5f` 或 `0.5F`）。
- `h` 或 `H` 后缀指定半精度浮点文字值（例如 `0.5h` 或 `0.5H`）。
- `bf` 或后缀指定大脑精度浮点文字值（例如 `0.5bf` 或 `0.5BF`）。
- 无符号整数文字的 `u` 或 `U` 后缀。
- 有符号长整数文字的 `l` 或 `L` 后缀。

光线追踪函数

原文：6.18 Ray Tracing Functions

所有操作系统：从 Metal 2.3 开始支持光线追踪功能。

光线追踪函数和类型在命名空间 metal::raytracing 的 中定义。在 Metal 2.3 中，它们仅在计算函数（内核函数）中受支持，除非下面另有说明。从 Metal 2.4 开始，顶点、片段和图块函数也支持它们。

6.18.1 加速结构函数​

在 Metal 2.3 中，调用以下函数之一来检查加速结构（参见第 2.17.6 节）是否为空：

bool is_null_primitive_acceleration_struct(primitive_acceleration_struct)

bool is_null_instance_acceleration_struct (instance_acceleration_struct ure)

在 Metal 2.4 中，调用以下函数来检查加速结构是否为空。

bool is_null_acceleration_struct(acceleration_struct)

在 Metal 3.1 中。Metal 支持使用以下函数迭代实例加速结构引用的加速结构：

调用以下函数查询实例加速结构中实例的数量。

uint get_instance_count() const

调用以下函数来检索实例加速结构中包含的实例所引用的加速结构。返回类型是第 2.17.6 节中定义的模板化类型。

模板 <类型名称... 交叉点标签> 加速结构< 交叉点标签...>

如果声明的返回类型与实例加速结构中包含的实例引用的加速结构类型不匹配，则结果未定义。不使用实例和/或基元运动标签的实例加速结构可以作为包含这些标签的加速结构类型返回。例如，没有任何运动（实例或基元）的实例加速结构可以返回为：

- 加速结构<实例>
- 加速结构<实例、实例运动>
- 加速结构<实例、原始运动>
- 加速结构<实例、原始运动、实例运动>

此功能允许开发人员在使用多个加速结构类型时避免为每组标签提供专用交叉器，但由于遍历不需要这些标签的加速结构而导致潜在的性能成本。

PrimitiveAccelerationStructure 通过声明 MTLAccelerationStructureGeometryFlagOpaque 来编码三角形是不透明还是非不透明。可以通过调用 intersector.force_opacity() 来覆盖不透明度。如果使用，它将优先于每个实例的不透明性标志（MTLAccelerationStructureInstanceFlagOpaque 和 MTLAccelerationStructureInstanceFlagNonOpaque），而后者又优先于每个几何体的不透明性。

6.18.5 间接实例描述符​

从 Metal 3.1 开始，开发人员可以从 GPU 填写间接实例描述符。Metal 提供了以下类型定义：

枚举 MTLAccelerationStructureInstanceOptions : uint { MTLAccelerationStructureInstanceOptionNone = 0, MTLAccelerationStructureInstanceOptionDisableTriangleCulling = (1 << 0),

MTLAccelerationStructureInstanceOptionTriangleFrontFacingWindingCounterClockwise = (1 << 1), MTLAccelerationStructureInstanceOptionOpaque = (1 << 2), MTLAccelerationStructureInstanceOptionNonOpaque = (1 << 3), };

typedef packed_float3 MTLPackedFloat3; typedef packed_float3 MTLPackedFloat4x3[4];

结构 MTLAccelerationStructureInstanceDescriptor { MTLPackedFloat4x3 conversionMatrix; MTLAccelerationStructureInstanceOptions 选项； uint 掩码； uint intersectionFunctionTableOffset； uint AccelerationStructureIndex； };

结构 MTLAccelerationStructureUserIDInstanceDescriptor { MTLPackedFloat4x3 conversionMatrix;

MTLAccelerationStructureInstanceOptions 选项； uint 掩码； uint intersectionFunctionTableOffset； uint AccelerationStructureIndex； uint 用户 ID； };

为了方便填写描述符，Metal 提供了从 Acceleration_struct 到 MTLResourceID 的隐式转换。

Acceleration_struct PrimitiveAStruct = ...; MTLResourceID 资源_id = PrimitiveAStruct;

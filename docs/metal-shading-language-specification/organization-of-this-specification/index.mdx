# 本规范的组织

<OrigninalPDF title="1.2 Organization of This Specification" />

本文档分为以下章节：

- 本章 "简介" 是对本文档的介绍，涵盖 Metal 和 C++14 之间的异同。它还详细介绍了 Metal 编译器的选项，包括预处理器指令、数学内在函数选项以及控制优化选项。
- "数据类型" 列出了 Metal 数据类型，包括表示向量、矩阵、缓冲区、纹理和采样器的类型。它还讨论了类型对齐和类型转换。
- "操作员" 列出了 Metal 操作员。
- "地址空间" 描述了用于分配具有访问限制的内存对象的不相交地址空间。
- "函数和变量声明" 详细介绍了如何声明函数和变量，以及指定限制的可选属性。
- "Metal 标准库" 定义了内置 Metal 函数的集合。
- "数值合规性" 描述了表示浮点数的要求，包括数学运算的准确性。

除非另有说明，自 Metal 1 起，iOS 和 macOS 支持本文档中描述的功能（函数、枚举、类型、属性或运算符）。

在本文档的其余部分中，缩写 X.Y 代表 "Metal 版本 X.Y"。例如，2.1 表示 Metal 2.1。请注意，虽然 MSL 着色语言支持某项功能，但并非所有 GPU 都支持该功能。请参阅 [developer.apple.com](https://developer.apple.com/) 上的 Metal 功能集表。

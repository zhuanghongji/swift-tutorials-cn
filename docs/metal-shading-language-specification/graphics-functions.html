图形函数

原文：6.10 Graphics Functions

本节及其子节中的图形函数在头文件 中定义。你只能从顶点函数或片段函数调用这些图形函数。

6.10.1 片段函数​

你只能在片段函数（请参阅第 5.1.2 节）内或从片段函数调用的函数内调用本节中的函数（在表 6. 18 、表 6. 19 和表 6. 20 中列出）。否则，行为未定义，并可能导致编译时错误。

可以创建片段函数辅助线程来帮助评估与片段线程一起使用的导数（显式或隐式）。片段函数辅助线程执行与非辅助片段线程相同的代码，但不会产生修改渲染目标或片段函数可以访问的任何其他内存的副作用。尤其：

  • 当片段函数执行完成而没有对渲染目标进行任何更新时，与辅助线程相对应的片段将被丢弃。- 由辅助线程执行的对缓冲区和纹理的存储和原子操作对与缓冲区或纹理关联的底层内存没有影响。

6.10.1.1 片段函数 - 导数​

Metal 包含表 6. 18 中的函数来计算导数。T 是 float、float2、float3、float4、half、half2、half3 或 half4 之一。

在非均匀控制流中导数是未定义的。

注意：在 Metal 2.3 之前，discard_fragment 可能会使控制流变得不均匀。从 Metal 2.3 开始，discard_fragment 不影响控制流是否被认为是非均匀的。有关详细信息，请参阅第 6.10.1.3 节。

表 6.18。Metal 标准库中的衍生片段函数​

内置片段函数描述

``` T dfdx(T p) 返回指定值相对于屏幕空间 x 坐标的高精度偏导数。````

``` T dfdy(T p) 返回指定值相对于屏幕空间 y 坐标的高精度偏导数。````

T fwidth(T p) 使用 p 的局部差分返回 x 和 y 的绝对导数之和；即 fabs(dfdx(p)) + fabs(dfdy(p))

6.10.1.2 片段函数 - 示例

Metal 包括表 6 中列出的每个样本函数。19. get_num_samples 和 get_sample_position 返回颜色附件的样本数以及给定样本索引的样本偏移量。例如，对于透明度超级采样，这些函数可用于对每个片段进行着色，但对每个样本进行 alpha 测试。

表 6.19。Metal 标准库中的示例片段函数​

内置片段函数描述

``` uint get_num_samples() 返回多重采样颜色附件的样本数。````

float2 get_sample_position(uint index)

``` 返回给定样本索引索引的归一化样本偏移量 (x, y)。x 和 y 的值在 [0.0 ... 1.0] 范围内。````

如果你有自定义样本位置（使用 MTLRenderPassDescriptor 的 setSamplePositions:count: 方法设置），则 get_sample_position(index) 返回为指定索引编程的位置。

6.10.1.3 片段函数 - 流控制​

表 6. 20 中的 Metal 函数终止一个片段。

表 6.20。Metal 标准库中的片段流控制函数​

内置片段函数描述

``` void Discard_fragment(void) 将当前片段标记为已终止，并丢弃片段函数的该片段的输出。````

在调用 discard_fragment 之前从片段线程写入缓冲区或纹理不会被丢弃。

与片段线程关联的多个片段线程或辅助线程一起执行以计算导数。在 Metal 2.3 之前，如果这些线程中的任何一个（但不是全部）执行 discard_fragment 函数，则该线程将终止，并且任何派生计算（显式或隐式）的行为都未定义。从 Metal 2.3 开始，discard_fragment 将片段标记为终止，同时继续并行执行，并且对是否定义导数没有影响。即使执行继续，写入行为仍与以前相同。在 discard_fragment 之后，片段将丢弃片段输出并丢弃所有对缓冲区或纹理的写入。

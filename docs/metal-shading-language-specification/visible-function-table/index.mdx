## 2.15 可见函数表

所有操作系统：自 Metal 2. 3 起支持类型。

在头文件中定义的 visible_function_table 类型用于表示存储在设备内存中的可见函数的函数指针表（参见第 5.1.4 节）。在 Metal 2.3 中，它可以在计算（内核）函数中使用。从 Metal 2.4 开始，它可以用于片段、顶点和图块函数。它是一种不透明类型，无法从 GPU 修改表的内容。 visible_function_table 类型可以在参数缓冲区中使用，或者使用缓冲区绑定点直接传递给限定函数。

使用模板参数 T 声明一个 visible_function_table 类型，其中

T 是表中存储的函数的签名，使用以下模板函数。

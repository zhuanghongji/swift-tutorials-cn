## 6.4 关系函数

表 6. 3 中的关系函数在头文件 中定义。 T 是标量或向量浮点类型之一，包括 bfloat 类型。 Ti 是标量或向量整数或布尔类型之一。 Tb 仅指标量或向量布尔类型。

## 表 6.3。 Metal 标准库中的关系函数

`内置关系函数说明`

``` bool all(Tb x) 仅当 x 的所有分量都为 true 时才返回 true。 ````

``` bool any(Tb x) 仅当 x 的任何组件为 true 时才返回 true。 ````

``` Tb isfinite(T x) 有限值测试。 ````

``` Tb isinf(T x) 测试无穷值（正或负）。 ````

``` Tb isnan(T x) 测试 NaN。 ````

``` Tb isnormal(T x) 测试正常值。 ````

``` Tb isordered(T x, T y) 测试参数是否有序。 isordered() 接受参数 x 和 y 并返回结果 (x == x) && (y == y)。 ````

``` Tb isunordered(T x, T y) 测试参数是否无序。 isunordered() 接受参数 x 和 y，如果 x 或 y 为 NaN，则返回 true；否则返回 false。 ````

``` Tb not(Tb x) 返回 x 的按组件逻辑补码。 ````

`Ti select(Ti a, Ti b, Tb c)`

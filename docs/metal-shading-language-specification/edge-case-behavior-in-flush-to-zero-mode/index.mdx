# 清零模式下的边缘情况行为

<ToBePolishedAfterTranslation />

<OriginalPDF title="7.5 Edge Case Behavior in Flush to Zero Mode" />

如果非规范化值刷新为零，则函数可能会返回四个结果之一：

1. 非清零模式的任何一致结果。
2. 如果步骤 1 给出的结果在舍入之前是次正规的，则可以将其刷新为零。
3. 如果一个或多个次正规操作数被刷新为零，则该函数的任何非刷新一致结果。
4. 如果步骤 3 的结果在舍入之前是次正规的，则结果可能会被刷新为零。

在上述每种情况下，如果操作数或结果刷新为零，则零的符号未定义。

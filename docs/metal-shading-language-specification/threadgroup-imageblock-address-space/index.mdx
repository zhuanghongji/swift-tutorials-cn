# threadgroup_imageblock 地址空间

<OrigninalPDF title="4.5 threadgroup_imageblock Address Space" />

`threadgroup_imageblock` 地址空间是指在线程组内存中分配的对象，只能使用 `imageblock<T, L>` 对象访问（请参阅第 2.11 节）。指向在 `threadgroup_imageblock` 地址空间中分配的用户定义类型的指针可以是图块着色函数的参数（请参见第 5.1.9 节）。每个图块只有一个线程组，每个线程组都可以访问线程组内存以及与其图块关联的图像块。

- 内核函数中 `threadgroup_imageblock` 地址空间中分配的变量是为执行内核的每个线程组分配的，由线程组中的所有线程共享，并且仅在执行内核的线程组的生命周期内存在。线程组中的每个线程都使用显式 2D 坐标来访问图像块。不要假设线程和图像块之间有任何特定的空间关系。线程组尺寸可以小于图块尺寸。

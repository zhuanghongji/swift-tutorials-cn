# 附加限制

<OrigninalPDF title="5.11 Additional Restrictions" />

MSL 函数和参数具有以下附加限制：

- 不保证从顶点函数写入缓冲区对于从给定基元的关联片段函数读取是可见的。
- 如果顶点函数确实写入一个或多个缓冲区或纹理，则其返回类型必须为 `void`。
- 顶点或片段函数的返回类型不能包含压缩向量类型、矩阵类型、结构类型、引用或类型指针的元素。
- 使用 `stage_in` 属性声明的片段函数的输入数量是有限的。不同功能集的输入限制有所不同。[Metal-Feature-Set-Tables.pdf](https://developer.apple.com/metal/Metal-Feature-Set-Tables.pdf) 上的 Metal 功能集表列出了“GPU 系列的实现限制”下方的具体限制。（一个输入向量算作 n 个输入标量，其中 n 是向量中的分量数。）
- 图形或内核函数的参数的参数类型不能是派生类。此外，使用 `stage_in` 属性声明的图形函数的参数类型不能是派生类。

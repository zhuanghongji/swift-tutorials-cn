## 概述

Swift Charts 是一个强大而简洁的 SwiftUI 框架，用于将数据转换为信息丰富的可视化（informative visualizations）。借助 Swift Charts，你可以使用最少的代码构建有效且可定制的图表。该框架提供标记（marks）、比例（scales）、轴（axes）和图例（legends）作为构建块（building blocks），你可以将它们组合起来开发各种数据驱动的图表。

<Figure src={require("./image_1.png")} size={680} />

你可以通过多种方式使用 Swift Charts 来传达数据中的模式（patterns）或趋势（trends）。你可以创建各种图表，包括折线图（line charts）、条形图（bar charts）和散点图（scatter plots），如上所示。当你使用此框架创建图表时，它会自动生成适合你的数据的比例和轴。

Swift Charts 支持本地化和辅助功能。你还可以使用图表修饰器覆盖默认行为以自定义图表。例如，你可以通过向图表添加动画来创建动态体验。

## 初始化器

### init(format:preset:position:values:stroke:)

```swift
// 使用给定属性创建轴标记，将覆盖默认标记。默认内容将用于轴标记
//
// - format: 用于标签的格式
// - preset: 轴标记的预设
// - position: 轴标记的位置
// - values: 轴标记的值
// - stroke: 用于网格线和刻度线的笔画
public init<Format>(format: Format,
                    preset: AxisMarkPreset = .automatic,
                    position: AxisMarkPosition = .automatic,
                    values: AxisMarkValues = .automatic,
                    stroke: StrokeStyle? = nil
) where Content == Never,
        Format : FormatStyle,
        Format.FormatInput : Plottable,
        Format.FormatOutput == String
```

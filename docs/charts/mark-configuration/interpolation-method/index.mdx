## 类型方法

### cardinal(tension:)

```swift
// 使用给定的张力参数，用基数样条插值数据点
///
// - tension: 控制基数样条线中切线长度的参数
@inlinable public static func cardinal(tension: CGFloat) -> InterpolationMethod
```

### catmullRom(alpha:)

```swift
// 使用给定的 alpha 参数，通过 Catmull-Rom 样条线对数据点进行插值
//
// - alpha: Catmull-Rom 样条线的参数。对于均匀样条线使用 0，对于向心样条线使用 0.5，对于弦样条线使用 1.0
@inlinable public static func catmullRom(alpha: CGFloat) -> InterpolationMethod
```

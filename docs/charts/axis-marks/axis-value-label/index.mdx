## 初始化器

### init(\_:centered:<wbr/>anchor:multiLabelAlignment:collisionResolution:offsetsMarks:orientation:horizontalSpacing:verticalSpacing:)

```swift
// 使用给定属性构造一个轴值标签以显示给定字符串
//
// - titleKey: 从本地化字符串生成的标题
// - centered: 是否使标签在两个轴值之间居中。如果 nil ，则对于离散数据默认为 true，对于连续数据默认为 false
// - anchor: 附加到 position 的文本元素边界框上的锚点
// - multiLabelAlignment: 沿轴的标签如何彼此对齐
// - collisionResolution: 如何解决与其他标签冲突的标签
// - offsetsMarks: 是否偏移标记以适应标签使用的空间
// - orientation: 标签的方向
// - horizontalSpacing: 标签的水平间距。如果 nil ，将使用默认间距
// - verticalSpacing: 标签的垂直间距
public init(_ titleKey: LocalizedStringKey,
            centered: Bool? = nil,
            anchor: UnitPoint? = nil,
            multiLabelAlignment: Alignment? = nil,
            collisionResolution: AxisValueLabelCollisionResolution = .automatic,
            offsetsMarks: Bool? = nil,
            orientation: AxisValueLabelOrientation = .automatic,
            horizontalSpacing: CGFloat? = nil,
            verticalSpacing: CGFloat? = nil
) where Content == Text
```

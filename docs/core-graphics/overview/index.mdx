# 概述

<OriginalLink
  title="Core Graphics"
  value="https://developer.apple.com/documentation/coregraphics"
/>

利用 [Quartz](https://developer.apple.com/documentation/quartz) 技术的强大功能以高保真输出执行轻量级 2D 渲染。处理基于路径的绘图、抗锯齿渲染、渐变、图像、颜色管理、PDF 文档等。

该文档集提供了 Quartz 框架的 API 参考；特别是对于 [Quartz Composer](https://developer.apple.com/documentation/quartz/quartz_composer)、[ImageKit](https://developer.apple.com/documentation/quartz/imagekit) 和 [PDFKit](https://developer.apple.com/documentation/quartz/pdfkit)。

- Quartz Composer API 支持处理和渲染图形数据，并允许开发人员为 Quartz Composer 开发工具创建自定义补丁。
- ImageKit 提供用户界面支持，用于浏览、编辑和保存图像、显示幻灯片以及浏览和预览 Core Image 滤镜。
- PDFKit 是一种允许应用程序显示和操作 PDF 文档的技术。

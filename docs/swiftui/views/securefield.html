SecureField

原文：SecureField

用户在其中安全地输入私人文本的控件。

概述​

在你希望行为类似于 TextField 但不希望用户的文本可见，请使用 SecureField。通常，你可以使用它来输入密码和其它敏感信息。

一个 SecureField 使用一个字符串值的绑定，以及一个用户提交编辑时执行的闭包，例如按 Return 键。该输入框在每次按键（keystroke）或其它编辑时更新绑定字符串，因此你可以随时从另一个控件（如 "完成" 按钮）读取其值。

下面的示例演示了一个绑定到字符串 password 的 SecureField。如果用户在 Secure Field 中提交编辑，则 onCommit 闭包会将 password 字符串发送到 handleLogin() 方法。

@State private var username = ""
@State private var password = ""

func handleLogin(username: String, password: String) {
    print(username, password)
}

var body: some View {
    VStack {
        TextField(
            "Username",
            text: $username
        )
        .autocapitalization(.none)
        .disableAutocorrection(true)
        .padding()
        .border(.separator)

        SecureField(
            "Password",
            text: $password
        ) {
            handleLogin(username: username, password: password)
        }
        .padding()
        .border(.separator)
    }
    .padding()
}

安全输入框提示​

可以向安全输入框提供显式提示，以指导用户应提供哪些文本。安全输入框显示的上下文决定了可以在何处以及何时使用提示和标签。例如，macOS 上的表单将始终将标签放在输入框的前缘旁边，并在可用时使用提示作为输入框本身内的占位符文本。在 iOS 上的相同上下文中，提示或标签将用作占位符文本，具体取决于是否提供提示。

@State private var username = ""
@State private var password = ""

var body: some View {
    VStack {
        Form {
            TextField(
              text: $username,
              prompt: Text("Required")
            ) {
                Text("Username")
            }
            SecureField(
              text: $password,
              prompt: Text("Required")
            ) {
                Text("Password")
            }
        }
    }
}

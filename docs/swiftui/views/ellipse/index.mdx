# Ellipse

<OriginalLink
  title="在包含它的视图 frame 内对齐的椭圆"
  value="https://developer.apple.com/documentation/swiftui/ellipse"
/>

```swift
@frozen public struct Ellipse : Shape {
    // ...

    /// Describes this shape as a path within a rectangular frame of reference.
    public func path(in rect: CGRect) -> Path

    /// Creates a new ellipse shape.
    @inlinable public init()
}
```

## 概述

圆的半径等于框架矩形最小边缘长度的一半，示例如下：

<ScreenshotBlock src={Ellipse1Image}>

```swift
VStack(spacing: 20) {
    Ellipse()
        .frame(width: 200, height: 200)
        .background(.gray.opacity(0.2))

    Ellipse()
        .fill(.orange)
        .frame(width: 300, height: 200)
        .background(.gray.opacity(0.2))

    Ellipse()
        .fill(.purple)
        .stroke(.green, lineWidth: 10)
        .frame(width: 200, height: 100)
        .background(.gray.opacity(0.2))
}

// 如右图，当 frame 宽高比相同时，Ellipse 变现为 "圆形"
```

</ScreenshotBlock>

跟之前谈到的 [Circle](../circle) 视图一样，`Ellipse` 也扩展了一个 `inset(by:)` 修饰器：

```swift
extension Ellipse : InsettableShape {

    /// Returns `self` inset by `amount`.
    @inlinable public func inset(by amount: CGFloat) -> some InsettableShape

    /// The type of the inset shape.
    public typealias InsetShape = some InsettableShape
}
```

示例如下：

<ScreenshotBlock src={Ellipse2Image}>

```swift
VStack(spacing: 20) {
    Ellipse()
        .fill(.orange)
        .frame(width: 300, height: 200)
        .background(.gray.opacity(0.2))

    Ellipse()
        // highlight-next-line
        .inset(by: 20)
        .fill(.orange)
        .frame(width: 300, height: 200)
        .background(.gray.opacity(0.2))

    Ellipse()
        // highlight-next-line
        .inset(by: 50)
        .fill(.orange)
        .frame(width: 300, height: 200)
        .background(.gray.opacity(0.2))
}
```

</ScreenshotBlock>

同样地，你也可以用扩展自 `Shape` 的 `size(_:)` 或 `size(width:height:)` 修饰器来约束包含 `Ellipse` 的 frame 尺寸：

<ScreenshotBlock src={Ellipse3Image}>

```swift
VStack(spacing: 20) {
    Ellipse()
        // highlight-next-line
        .size(width: 100, height: 50)
        .fill(.orange)
        .frame(width: 300, height: 200)
        .background(.gray.opacity(0.2))

    Ellipse()
        // highlight-next-line
        .size(CGSize(width: 150.0, height: 75.0))
        .fill(.orange)
        .frame(width: 300, height: 200)
        .background(.gray.opacity(0.2))

    Ellipse()
        // highlight-next-line
        .size(CGSize(width: 250.0, height: 125.0))
        .fill(.orange)
        .frame(width: 300, height: 200)
        .background(.gray.opacity(0.2))
}
```

</ScreenshotBlock>

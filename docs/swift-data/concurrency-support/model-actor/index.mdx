# ModelActor

<OriginalSource
  title="ModelActor"
  url="https://developer.apple.com/documentation/swiftdata/modelactor"
  summary="用于提供对一致模型（conforming model）的属性的互斥（mutually-exclusive）访问的接口"
/>

```swift
@available(swift 5.9)
@available(macOS 14, iOS 17, tvOS 17, watchOS 10, *)
public protocol ModelActor : Actor {
    // ...
}
```

## 访问执行者

### unownedExecutor

```swift
// 对模型 actor 执行者的优化的、无主的（unonwned）引用
nonisolated public var unownedExecutor: UnownedSerialExecutor { get }
```

## 实例属性

### modelContainer

```swift
// ModelActor 的管理应用程序的架构和模型存储配置的容器
nonisolated var modelContainer: ModelContainer { get }
```

### modelContext

```swift
// 在模型 actor 上序列化任何运行代码的上下文
public var modelContext: ModelContext { get }
```

### modelExecutor

```swift
// 协调模型 actor 访问的执行器
nonisolated var modelExecutor: ModelExecutor { get }
```

:::caution

不要使用执行器访问模型上下文。请改用 `context` 属性。

:::
